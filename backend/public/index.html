Backend Dev Project

Endpoints

  • /login - Login endpoint for Spotify
  • /callback - Callback for Spotify login
  • /refresh_token - To refresh Spotify token
  • /searchitem/:searchitem - Look for Spotify track (authentication needed)
  • /searchfavorites/:favoriteids - To get track data of list of trackid's
  • /getaccess - get the accesstoken
  • /mongodb/:userid" - get specific user data
  • /mongoAddFavorite/ - add a new track to users' favorites
  • /mongoFavorites/ - get all favorited tracks from user
  • /mongoDelete/ - remove a track from a users' favorites
  • /dominantcolor/:imagelink - get the dominant color out of an image
